# Visual Identity

QuadBoard is an infrastructure dashboard.

It should feel:

- Professional
- Minimal
- Fast
- Modern
- Trustworthy

The interface should never compete with the applications it exposes.

QuadBoard is the entry point, not the destination.

## Logo

The logo consists of four connected quadrants representing the main domains of a self-hosted infrastructure.

- Applications
- Routing
- Security
- Infrastructure

The central node represents QuadBoard itself.

## Variants

- Full logo
- Icon only
- Monochrome
- Dark mode
- Light mode

## Usage

Always keep enough spacing around the logo.

Do not distort it.

Do not recolor the logo outside of the official palette.
